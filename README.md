# Task Manager

A simple and responsive **Task Manager** built with **HTML, CSS, and JavaScript**. This project allows users to create, organize, move, and delete tasks using a drag-and-drop interface. Tasks are automatically saved in the browser using **Local Storage**, ensuring data persists even after refreshing the page.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- HTML5 Drag & Drop API

---

## Features

- Add new tasks
- Delete tasks
- Drag and drop tasks between columns
- Automatic task count for each column
- Persistent task storage using Local Storage
- Responsive and clean user interface
